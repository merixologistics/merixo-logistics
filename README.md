# Global Export Logistics Website

A premium, animated logistics website designed for a company specializing in exporting coconuts and spices.

## Features

- **Modern Design**: Built with React and Tailwind CSS, utilizing a custom color palette inspired by nature (Coconut Green, Spice Red/Gold).
- **Animations**: Smooth scroll reveals and interactions powered by Framer Motion.
- **Responsive**: Fully responsive layout for Desktop, Tablet, and Mobile.
- **Sections**:
  - **Hero**: Cinematic introduction with video/image background.
  - **Services**: Cards detailing Supply Chain, Warehousing, and Freight services.
  - **Products**: Spotlight on Coconuts and Spices with visual richness.
  - **Contact**: Functional form layout and contact details.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Google Fonts (Outfit)

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```
